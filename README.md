## Getting started
* Clone repository
```
git clone git@gitlab.asoft-python.com:hoa.nguyen/practice-javascript.git
```

* Checkout branch
```
git checkout practice-JS
```

* Change directory to practice-JS folder
```
cd practice-js
```

* Run application
```
npm start
```

* Open browser with the link below
```
http://localhost:1234/login.html
```
